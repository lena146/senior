※ 이 폴더에서 웹사이트를 여는 방법

[가장 쉬운 방법] 서버_켜고_브라우저_열기.command 더블클릭
  → "서버_켜고_브라우저_열기.command" 파일을 더블클릭하세요.
  → 터미널 창이 뜨고, 잠시 후 브라우저가 자동으로 열립니다.
  → 처음 실행 시 "확인되지 않은 개발자"라고 나오면:
     파일을 우클릭 → "열기"를 선택해서 한 번 열어 주세요.
  → 서버를 끄려면 터미널 창을 닫거나 Ctrl+C 하세요.

[다른 방법 1] index.html 더블클릭
  → "index.html" 파일을 더블클릭하면 브라우저에서 바로 열립니다.
  → (서버 없이 파일로 여는 방식)

[다른 방법 2] 터미널에서 직접 서버 실행
  → 터미널을 열고 아래를 한 줄씩 입력한 뒤 엔터:
     cd /Users/wonmi/senior2/standalone
     python3 -m http.server 8080
  → 브라우저 주소창에 입력: http://localhost:8080
  → 끌 때: 터미널에서 Ctrl+C
