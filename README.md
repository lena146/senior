# 시니어 여행 플래너

시니어의 **건강 상태**, **여행 성향**, **선호 여행지·기간**을 반영해 여행 정보를 큐레이션하고, 맞춤 일정을 만들어 주는 웹 서비스입니다. 모바일 최적화되어 있습니다.

## 기능 요약

### 1) 여행 정보 큐레이션
- 다양한 출처(블로그·유튜브·공공데이터)를 연상한 **숙소·교통·관광지·음식** 정보 수집·선별
- **Top 3 요약**과 카테고리별 **카드/리스트** 형태로 정리
- 비교·결정 과정 단순화 (요약 정리 제공)

### 2) 건강·성향 맞춤 일정 플래너
- **건강 데이터**(운동량, 휴식 필요도, 질환)와 **여행 성향**(활동/휴양/문화)을 프로필에 반영
- 맞춤형 **일정 자동 생성** 및 **난이도·페이스 조정**
- **일정 다시 만들기**로 재생성 지원

### 3) 프로필 설문 (아이콘 활용)
- 운동량·휴식 필요도·건강 이슈·여행 스타일을 **간단한 일러스트 아이콘**으로 선택
- 시니어가 선택하기 쉬운 큰 터치 영역, 단계별 진행

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

## 빌드

```bash
npm run build
npm run preview   # 빌드 결과 미리보기
```

## 기술 스택

- **React 18** + **Vite**
- **React Router** (라우팅)
- **Context API** (프로필 상태)
- **CSS Modules** (모바일 우선 반응형)

## 프로젝트 구조

```
src/
  components/   # Layout, Icons
  context/      # ProfileContext
  data/         # curationData, plannerLogic
  pages/        # Home, Survey, Curation, Planner
  App.jsx, main.jsx, index.css
```

## 사용 흐름

1. **홈** → **나의 프로필**에서 건강·성향·목적지·기간 선택 (아이콘/칩 선택)
2. **여행 정보**에서 해당 지역 숙소·교통·관광·맛집 Top 3 및 상세 보기
3. **맞춤 일정**에서 프로필 기반 자동 일정 확인 후, 필요 시 "일정 다시 만들기"로 재생성
